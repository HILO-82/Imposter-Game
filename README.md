# Imposter-Game
Simple HTML File that plays Imposter

________________________________________________________________________________________________________________________________________________

**How to Run the code:**

Run the App.py file

then Click the link at the bottom the file to Access the game

You can have up to 13 players with up to 4 imposters and 2 jesters


________________________________________________________________________________________________________________________________________________

**Roles Explaination:**

Crewmate: They get given the word and are tyring to find out and vote out who the imposter is. 

Jesters: these players are trying to convince the other players that they are the imposter, if they get voted out the win. 

Imposters: they are trying to guess the word, if they believe they know the word they can reveal themselves and attempt to guess the word to win. otherwise they are simply trying to not be voted out.
